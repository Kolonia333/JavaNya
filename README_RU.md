# НяДжава (JavaNya)

Транслятор эзотерического языка программирования на основе Java с синтаксисом из вселенной Steins;Gate.

## Требования
- JDK 17+
- Gradle

## Запуск
```
gradlew run
```
Введи путь к `.jnya` файлу когда программа попросит.
