##  О проекте

Этот проект чистый фан и дань уважения вселенной *Steins;Gate*. Мы переписали стандартные ключевые слова Java так, чтобы каждая строчка кода приближала тебя к открытию Врат Штейна.

* Полная замена стандартных ключевых слов Java на термины из Steins;Gate.
* Поддержка базовых структур: циклы, коллекции, многопоточность.

##  Установка и Запуск
1.Скачайте все файлы через GitHub 
или
скопируй эту команду и вставь её в терминал:
```bash
git clone https://github.com/Kolonia333/JavaNya.git
```
2. запусти скрипт и укажи путь к тому файлу который ты хочешь инициализировать 
---
### Важно 
Если вместо текста в консоли вы видите странные символы:
1. Используйте **PowerShell** вместо **cmd**.
2. Перед запуском введите команду: `chcp 65001`.
3. Если это не помогло, самый надежный способ запустить проект через любую IDE, там кодировка UTF-8 должна быть настроена автоматически.
