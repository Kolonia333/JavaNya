# JavaNya (ニャジャワ) - 日本語ドキュメント

『STEINS;GATE (シュタインズ・ゲート)』の世界観に基づく構文を持つ、Javaベースの難解プログラミング言語 (Esoteric Programming Language) トランスレータ。

## 推奨環境
- JDK 
- Gradle

プログラムの指示に従って、`.jnya` ファイルのパスを入力してください。

## 構文 (シンタックス)

すべてのファイルは `Тунтуру!` (トゥゥントゥルー！) で始まる必要があります。
