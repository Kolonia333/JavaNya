# JavaNya（НяДжава）

Steins;Gateの世界観をベースにしたJava言語のエソテリックプログラミング言語トランスレーターです。

## 必要環境
- JDK 17以上
- Gradle

## 実行方法
```
gradlew run
```
プロンプトが表示されたら`.jnya`ファイルのパスを入力してください。

## 構文

すべてのファイルは`Тунтуру!`で始まる必要があります。
