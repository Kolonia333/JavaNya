# JavaNya (НяДжава)

An esoteric programming language based on Java with syntax from the Steins;Gate universe.

## Requirements
- JDK 17+
- Gradle

## Running
```
gradlew run
```
Enter the path to a `.jnya` file when prompted.

## Syntax

Every file must start with `Тунтуру!`
